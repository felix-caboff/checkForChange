====================
INSTALL REQUIREMENTS
====================

This is the specific setup for Debian 12.  There are some dependency weirdities.  The below commands "should" work.... You will need to:

PyEnv dependancies
==================

sudo apt install -y build-essential libbz2-dev libncursesw5-dev libreadline-dev libsqlite3-dev libssl-dev libffi-dev liblzma-dev zlib1g-dev tk-dev curl xz-utils wget libdbus-1-dev

PyEnv
=====
This can be done with:

curl -fsSL https://pyenv.run | bash
echo 'export PYENV_ROOT="$HOME/.pyenv"\n[[ -d $PYENV_ROOT/bin ]] && export PATH="$PYENV_ROOT/bin:$PATH"\neval "$(pyenv init - bash)"' >> ~/.bashrc


Python
======
The correct version of python can be added to this venv via PyEnv thusly:

# cd [SCRIPT DIRECTORY]
pyenv install 3.11


PipEnv
======

sudo apt install pipenv


Python libraries
================

pipenv install -r requirements.txt
